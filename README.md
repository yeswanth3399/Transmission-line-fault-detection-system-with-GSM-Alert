# Transmission Line Fault Detection with GSM Alert

## Overview

Transmission Line Fault Detection with GSM Alert is an Arduino-based embedded system project designed to detect faults occurring in overhead or transmission power lines and send instant SMS notifications using a GSM module.

The system continuously monitors different fault conditions such as:

- Short Circuit Fault
- Open Circuit Fault
- High Voltage Fault
- Fire Detection

Whenever a fault is detected:

- The fault type is displayed on a 16x2 LCD.
- A buzzer alert is activated.
- An SMS alert is sent automatically to the authorized mobile number.

This project helps improve fault response time and power system reliability.

# Fault Conditions

## Short Circuit Fault
Detects abnormal direct connection between conductors.

## Open Circuit Fault
Detects line disconnection or broken path.

## High Voltage Fault
Detects voltage levels exceeding safe operating conditions.

## Fire Detection
Detects fire hazards near the transmission line.

# Working Principle

1. Arduino continuously monitors all fault inputs.
2. If any fault occurs:
   - LCD displays the detected fault.
   - Buzzer turns ON.
   - GSM module sends an SMS alert.
3. If no fault is detected:
   - LCD displays line status as stable.
   - Voltage value is shown.
